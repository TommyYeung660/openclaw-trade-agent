## (3) Paper Digest（量化交易/風控/因子/LLM in finance）

### 1. TradeFM: A Generative Foundation Model for Trade-flow and Market Microstructure
**arXiv ID**: 2602.23784  
**類別**: q-fin.TR, cs.LG, cs.AI

**研究問題**：如何建立一個可跨資產泛化的市場微觀結構基礎模型，從数十億筆交易事件中學習通用表徵

**方法**：開發 TradeFM（524M 參數生成式 Transformer），通過尺度不變特徵和通用 token 化方案，將異構多模態訂單流映射為統一離散序列，整合確定性市場模擬器

**數據/實驗設定**：在超過 9000 檔股票的数十億筆交易事件上訓練；生成的 rollouts 重現了金融回報的關鍵風格化特徵（厚尾、波動率聚集、回報無自相關）；在 APAC 市場上進行零樣本泛化測試

**主要結論**：TradeFM 相比 Compound Hawkes 基線模型實現 2-3 倍更低的分佈誤差，能夠 zero-shot 泛化到地理上 OOD 的 APAC 市場，複雜度僅輕微下降

**可落地啟示**：尺度不變的交易表徵捕捉市場微觀結構中的可遷移結構，為合成數據生成、壓力測試、基於學習的交易代理開闢新路徑

**限制/風險**：基於歷史訂單流的生成式模型可能無法完全適配未曾見過的市場制度變遷

### 2. LLM as a Risk Manager: LLM Semantic Filtering for Lead-Lag Trading in Prediction Markets
**arXiv ID**: 2602.07048  
**類別**: q-fin.RM, q-fin.ST

**研究問題**：預測市場中事件級時間序列與自然語言描述直接關聯，但發現穩健的領先-滯後關係因統計偽相關而具挑戰性

**方法**：雙階段因果篩選器：(i) 統計階段：使用 Granger 因果關係識別候選 leader-follower 對；(ii) LLM 語義階段：基於事件描述評估提出的因果方向是否具有合理的經濟傳導機制，重新排序候選對

**數據/實驗設定**：Kalshi Economics 市場的隱含機率時間序列；使用固定信號觸發交易協議將關係品質映射為已實現盈虧

**主要結論**：混合方法持續優於統計基線。滾動評估中勝率從 51.4% 提升至 54.5%；虧損交易平均金額從 649 USD 大幅降至 347 USD

**可落地啟示**：LLM 充當統計發現之上的語義風控層，優先選擇在變動市場條件下能泛化的領先-滯後關係

**限制/風險**：因果地面真值不可觀測，評估依賴於交易盈虧；方法有效性可能受市場領域和語言模型偏見影響

### 3. FinBloom: Knowledge Grounding Large Language Model with Real-time Financial Data
**arXiv ID**: 2502.18471  
**類別**: q-fin.ST, cs.LG, cs.AI, cs.CL

**研究問題**：LLMars 在生成人類響應方面出色，但難以處理需要即時資訊（如新聞、價格變動）的互動式金融查詢

**方法**：開發 Financial Agent（知識增強方法），通過三項貢獻實現：(i) 構建超過 50,000 個金融查詢與所需上下文的 Financial Context Dataset；(ii) 在 1400 萬篇路透社和 DPA 財經新聞，加上 12 萬篇 SEC 檔案的 25% 隨機樣本上微調 Bloom 7B，得到 FinBloom 7B；(iii) 在 Financial Context Dataset 上微調以作為 Financial Agent

**數據/實驗設定**：50,000+ 財經查詢配對上下文數據集；14M 財經新聞 + 3M SEC 檔案用於預訓練；FinBloom 7B（7B 參數）

**主要結論**：該代理生成相關金融上下文，實現高效即時數據檢索以回答用戶查詢，降低延遲並消除手動提供準確數據的需求

**可落地啟示**：使 LLMs 更好地處理動態金融任務，為即時金融決策、算法交易等任務提供流線化支持

**限制/風險**：依賴於外部數據源的可靠性和延遲；金融上下文數據集的覆蓋範圍可能限制了處理特殊查詢的能力

### 4. Betting under Common Beliefs: The Effect of Probability Weighting
**arXiv ID**: 2602.24194  
**類別**: q-fin.RM, econ.TH

**研究問題**：引入 Rank-Dependent Utility (RDU) 代理到無總體不確定性的 vNM 純交換經濟中，對 Pareto 最優配置的影響

**方法**：理論分析，展示 RDU 代理的機率加權函數（可視為不被財富邊際效度捕捉的機率風險厭惡）如何在共同基線信念下導致內生投注的 Pareto 最優點

**數據/實驗設定**：理論框架分析，無實證數據

**主要結論**：機率加權可充當內生信念異質性的來源，為共同信念與投機行為共存提供新的行為學基礎；仁慈社會規劃者可通過代價統計或金融教育來「推動」RDU 代理接近 vNM 行為，從而（部分）恢復完全保險配置的最優性

**可落地啟示**：機率加權函數在資產定價和投資者行為建模中的重要性，為行為金融提供理論支持

**限制/風險**：純理論分析，實證驗證困難；不同文化和市場環境下的 RDU 函數形狀可能異於文中假設

### 5. Combining Value-at-Risk and Expected Shortfall forecasts via Model Confidence Set
**arXiv ID**: 2406.06235  
**類別**: q-fin.RM

**研究問題**：尾風險度量（VaR、ES）預測準確性受多重不確定性來源（模型誤設定、數據限制、估計程序）影響

**方法**：基於 Model Confidence Set (MCS) 方法論的創新預測組合策略，使用嚴格一致的聯合 VaR-ES 損失函數識別最佳表現模型（構成優越模型集 SSM），將 SSM 中模型的 VaR 和 ES 預測使用不同加權方案組合

**數據/實驗設定**：9 個股票市場指數在 2.5% 和 1% 風險水平的數據

**主要結論**：所提出的組合預測器是預測尾風險度量的穩健替代方案，成功通過標準回測，且一致進入 MCS 的 SSM

**可落地啟示**：模型組合方法可顯著提升風險管理系統的穩健性，特別是對高度監管且需嚴格回測的金融機構

**限制/風險**：依賴於單個模型的品質和 MCS 設定假設；組合權重方案可能需要根據市場制度進行調整

### 研究想法總結

1. **微觀結構模型跨資產泛化驗證**：基於 TradeFM 的結果，可在美股以外的 APAC 或歐洲市場上驗證尺度不變交易表徵的泛化能力，構建多市場微觀結構共享表徵庫，提升跨市場套利或統計套利的信號質量。

2. **LLM 語義風控層集成到現有信號管道**：將 LLM 語義篩選集成到統計領先-滯後信號生成中，在策略回測時加入語義一致性檢查，過濾缺乏經濟傳導機制的統計偽相關關係，預期可降低策略在 OOD 時期的大幅虧損。

3. **金融增強 LLM 在即時查詢中的應用**：基於 FinBloom 的金融 Agent 模式，可在現有量化分析管道中加入 LLM 模塊，用於動態事件檢索、新聞摘要、風險事件分類，減少人工監控負擔並提升策略對新聞事件的反應速度。

4. **機率加權效用函數在投資者行為建模中的應用**：將 RDU 或機率加權函數引入投資者異質信念建模框架，用於解釋市場上的投機行為、資源配置異常，或用於構建行為一致性的風險管理場景。

5. **MCS 模型組合策略提升風險管理穩健性**：在多個 VaR/ES 模型預測的場景中，引入 MCS 框架識別優越模型集並組合預測，可作為量化風控系統的後處理層，減少單一模型失效帶來的系統性風險。
