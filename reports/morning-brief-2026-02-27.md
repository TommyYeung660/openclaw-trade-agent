### 1. Pools as Portfolios: Observed arbitrage efficiency & LVR analysis of dynamic weight AMMs
**來源**：q-fin.TR / q-fin.PM | 連結：https://arxiv.org/abs/2602.22069

**研究問題**：動態權重自動做市商（AMM）能否作為代幣化基金的有效執行層？

**方法**：
- 對 QuantAMM 協議上的兩個實時池（Ethereum 主網 + Base）進行區塊級套利分析
- 使用 Loss-vs-Rebalancing (LVR) 和 Rebalancing-vs-Rebalancing (RVR) 基準測量長期表現
- 時窗：2025 年 7 月 + 2026 年 1 月（兩次調倉窗）

**數據/實驗設定**：區塊級交易數據（Ethereum + Base），實時套利者行為追蹤

**主要結論**：
- 主網上：再平衡效率隨時間顯著提升（套利交易更頻繁、單次提取值更低），達到或超越 CEX 風格再平衡
- Base 鏈上：即使單次套利提取接近或低於零，再平衡仍持續，效率達到或超越「完美再平衡」LVR 基準
- 結論證明：動態權重 AMM 在 L2 上因路由與更低數據成本壓縮套利價差，表現優異

**可落地啟示**：
- **DeFi 投資組合管理**：AMM 可替代傳統 CEX 再平衡，降低滑點與執行成本
- **L2 優化**：鏈上低成本與路由驅動是代幣化基金執行的關鍵
- **風險管理**：LVR/RVR 基準可用於評估實際再平衡損失，優化池設計

**限制/風險**：
- 結果依賴特定協議（QuantAMM）與市場環境
- 套利者行為與 MEV 壓力相關，壓力下降可能影響效率
- Gas 費波動性可能衝擊執行成本

### 2. On the Estimation of Own Funds for Life Insurers: A Study of Direct, Indirect, and Control Variate Methods in a Risk-Neutral Pricing Framework
**來源**：q-fin.RM | 連結：https://arxiv.org/abs/2511.04412

**研究問題**：壽險中性定價框架下，直接法、間接法與控制變量法的實際表現比較？

**方法**：
- 理論證明直接法與間接法收斂至相同值
- 引入 2^T 混合估計器家族（包含直接法/間接法為邊界情況）
- 擴展至含市場摩擦的設定（現實應用）

**數據/實驗設定**：三個壽險資產負債管理模型（資產-負債耦合程度不同）

**主要結論**：
- 直接法與間接法理論等價，但實際表現受資產-負債耦合強度驅動
- 耦合現實設定中，耦合較強時間接法優於直接法
- 控制變量法可將方差降低至標準直接法的 1/10，但效取決於模型耦合度

**可落地啟示**：
- **壽險 SCR 計算**：耦合強時優先間接法；耦合弱時直接法可接受
- **控制變量法**：顯著降低方差，適用於高維度模擬場景
- **模型選型**：資產-負債耦合度是選擇估計器的關鍵指標

**限制/風險**：
- 控制變量法效取取決於模型特定結構，無通用優勢
- 市場摩擦引入增加模型複雜度
- 擴展至其他壽險類型需進一步驗證

### 3. A Bayesian approach to out-of-sample network reconstruction
**來源**：q-fin.ST | 連結：https://arxiv.org/abs/2602.21869

**研究問題**：如何利用歷史網絡快照預測未來網絡結構，並量化不確定性？

**方法**：
- 使用單參數適配度模型（fitness model）推斷鏈接機率
- 貝氏推斷：利用過去快照構建先驗，預測後續快照
- 每個預測快照成為下一個快照的可靠先驗（自持續機制）

**數據/實驗設定**：電子銀行間貸款市場（e-MID）1999-2012 年數據

**主要結論**：
- 準確恢復後續時間點的銀行連接數量，優於概率性基準
- 每個預測快照提供可靠先驗，實現樣本外重建
- 適合度模型簡單高效，僅需節點強度作輸入

**可落地啟示**：
- **銀行網絡風險管理**：預測潛在系統性連接變化，壓力測試
- **交易對手風控**：重建交易對手網絡，識別潛在關鍵節點
- **宏觀網絡分析**：適用於其他金融網絡（如支付網絡、股權相關性網絡）

**限制/風險**：
- 適合度模型假設網絡演化規律性，突發結構變化可能失效
- 僅適用於無向網絡，有向網絡需擴展
- 預測不確定性量化需進一步研究

### 4. When Fusion Helps and When It Breaks: View-Aligned Robustness in Same-Source Financial Imaging
**來源**：q-fin.ST / cs.LG | 連結：https[://arxiv.org/abs/2602.11020

**研究問題**：同源多視圖學習中，早融合（early fusion）與晚融合（late fusion）何時有效，對抗魯棒性如何？

**方法**：
- 兩個視圖：OHLCV K線圖（ohlcv）+ 技術指標矩陣（indic）
- 標籤定義：ex-post 最小移動閾值 tau（基於實現次日日收益絕對值）
- 對抗魯棒性：像素空間 L-infinity 攻擊（FGSM/PGD），視圖受限 vs 聯合威脅
- 時序分割：leakage-resistant time-block splits with embargo

**數據/實驗設定**：未公開金融圖像數據集（多視圖、對齊窗口）

**主要結論**：
- 融合有效取決於標籤穩定性：早融合在噪音設定下可負轉移；晚融合為可靠預設
- 對抗魯棒性急劇下降：微小預算下即失效
- 晚融合有助於視圖受限攻擊，但聯合攻擊仍具挑戰性
- 標籤設計關鍵：需考慮日收益噪音帶來的標籤歧義

**可落地啟示**：
- **金融圖像預測**：優先晚融合架構，避免負轉移
- **對抗訓練**：L-infinity 攻擊揭露視圖特定弱點，需魯棒訓練
- **標籤設計**：使用 ex-post 閾值過濾噪音樣本，提升模型穩定性

**限制/風險**：
- 結果依賴特定數據集，泛化至其他數據需驗證
- 聯合攻擊模型強大，需更強魯棒機制
- 標籤設計涉及實際收盤絕對值，可能引入前瞻偏差

### 5. Latent Context Compilation: Distilling Long Context into Compact Portable Memory
**來源**：cs.LG | 連結：https://arxiv.org/abs/2602.21221

**研究問題**：如何將長語境蒸餾成緊湊可移植記憶，提升 LLM 部署效率？

**方法**：
- 使用 LoRA 模組作為編譯器，將長語境蒸餾成緩湊 buffer tokens
- Self-aligned 優化策略：無需合語境相關 QA 對對，使用隨機查詢正則化
- Buffer tokens 可插入凍結基模型，無需修改權重

**數據/實驗設定**：Llama-3.1-8B，16x 壓縮比

**主要結論**：
- 16x 壓縮比下仍保留細節細節與推理能力
- Self-aligned 優化消除合成 QA 對對需求
- Buffer tokens 與凍結模型無縫兼容，狀態less portable

**可落地啟示**：
- **金融資訊檢索**：將長文檔/研報蒸餾成 buffer tokens，提升檢索效率
- **多輪模型部署**：避免多輪狀態，降低部署複雜度
- **成本優化**：壓縮長語境降低 API 調用成本

**限制/風險**：
- 壓縮比越高，資訊損失風險越大
- LoRA 訓練需額外計算
- Buffer tokens 插入效果取決於基模型能力

### 6. ACAR: Adaptive Complexity Routing for Multi-Model Ensembles with Auditable Decision Traces
**來源**：cs.LG | 連結：https://arxiv.org/abs/2602.21231

**研究問題**：如何在不犧牲可審計性的前提下，為多模型集成自適應路由？

**方法**：
- 使用自洽性方差（sigma，N=3 個測樣本）路由任務至單模型/雙模型/三模型執行模式
- 系統基於 TEAMLLM（不可變工件與完整決策蹤徑）
- 評估：MathArena, Reasoning Gym, LiveCodeBench, SuperGPQA（1,510 任務）

**數據/實驗設定**：Claude Sonnet 4, GPT-4o, Gemini 2.0 Flash（>7,550 次審計運行）

**主要結論**：
- Sigma 路由達到 55.6% 準確率，優於雙模型基準（54.4%），同時在 54.2% 任務上避免全集成
- 路由機制模型無關，無需學習組件
- 負面結果：
  - 檢索增強降低準確率 3.4 個百分點（語義不對齊引入噪音）
  - 模型一致但錯誤時（sigma=0），無法集成後修正
  - 基於代理信號（回應相似度/熵）的歸因估計與 leave-one-out 值弱相關

**可落地啟示**：
- **量化策略選型**：使用 Sigma 路由動態選擇模型，平衡成本與準確性
- **可審計性**：決策蹤徵完備記錄，合規要求
- **多模型集成優化**：避免低效檢索，避免過度依賴單一模型

**限制/風險**：
- Sigma 路由對任務分佈敏感
- 負面結果顯示檢索與歸因估計挑戰大
- 需額外 N=3 樣查成本

### 7. Neural network optimization strategies and the topography of the loss landscape
**來源**：cs.LG / stat.ML | 連結：https://arxiv.org/abs/2602.21276

**研究問題**：SGD 與擬牛頓法（quasi-Newton）在損失地形探索上有何差異？

**方法**：
- 對比 SGD（僅梯度）與擬牛頓法（曲率資訊 + Golden Section Search）
- 使用 kernel PCA 與 FourierPathFinder（損失地形低高度路徑演算法）
- 對比解的隔離度與泛化能力

**數據/實驗設定**：標準神經網絡任務（早停保證測試表現）

**主要結論**：
- SGD 解之間隔離度較低，擬牛頓解更孤立、更深
- 擬牛頓解在訓練集上表現更佳，但泛化更差
- SGD 探索平滑吸引盆地，擬牛頓尋找更深、更孤立的極小值
- 優化策略深刻影響解的性質與泛化能力

**可落地啟示**：
- **量化模型訓練**：SGD 通常泛化更優，擬牛頓法可能過擬合
- **損失地形分析**：可作為模型診斷工具，識別過擬合風險
- **優化器選擇**：泛化需求高時優先 SGD；追求訓練集表現時考慮擬牛頓法

**限制/風險**：
- 結果任務特定，泛化需驗證
- 擬牛頓法計算成本高
- 解的性質取決於損失地形結構
